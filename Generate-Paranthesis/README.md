# [GENERATE PARANTHESUS](https://leetcode.com/problems/generate-parentheses/)

### Difficulty: ***Medium***

Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.

```
Example 1:

Input: n = 3
Output: ["((()))","(()())","(())()","()(())","()()()"]
```
```
Example 2:

Input: n = 1
Output: ["()"]
```

Time COmplexity: O(n2)
